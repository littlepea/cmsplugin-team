=====================
Django CMS team plugin
=====================

Django CMS plugin to list team members/employees.

Installation
------------

1. Install with pip::

    pip install cmsplugin-team

2. Add `filer` and `cmsplugin_team` to your INSTALLED_APPS in `settings.py`::

    INSTALLED_APPS = (
        ...
        'filer',
        'cmsplugin_team',
    )
